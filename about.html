Bienvenue sur ce blog

Je vous invite à découvrir JEKYLL qui est un générateur de site statique Open Source écrit en language Ruby dévéloppé par
Tom Preston-Werner, le fondateur de Github. Il est devenu très populaire parmi les générateurs de sites statiques pour sa mise en service et son fonctionnement rapide.


Générateur de "site statique" ?

En effet, en quelques minutes vous pouvez disposer d'un blog responsive et vous consacrer de suite à la saisie de vos "posts" sans difficulté.
De plus, vous bénéficiez d'un hébergement internet gratuit avec Github.


Thèmes gratuits

La communauté "JEKYLL" étant assez importante il vous est possible de profiter de la mise à disposition gratuite de nombreux thèmes.
Et après quelques modifications votre propre "blog" sera opérationnel.


Hébergement, outils

Comme indiqué ci-dessus Github vous offre la gratuité d'hébergement de votre blog.
Nous verrons aussi comment dialoguer entre votre "blog" installé sur Github et votre configuration locale avec Github Desktop
Il vous est aussi possible d'héberger gratuitement votre blog sur un autre serveur et de configurer celui-ci avec un nom de domaine que vous aurez acquis par ailleurs.


Alors ! Si vous êtes intéressé(e) .... c'est parti ...
